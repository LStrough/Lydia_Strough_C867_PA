TITLE: Class Roster (Scripting and Programming - Applications, C867 PA) 

PURPOSE: Operational class roster management program with the ability to add, remove, and print all students. Additional abilities include the capacity to print students' average course completion time, populate invalid student emails, and categorize students by degree program. Developed in Xcode and written in C++.

APPLICATION VERSION: Application Version 1.0, 06/27/2021

IDE: Xcode
Programming Language: C++
