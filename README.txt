TITLE: Class Roster (Scripting and Programming - Applications, C867 PA) 

PURPOSE: A C++ program that simulates a class roster management program with the ability to add, remove, and print all students. The program can also print a student's average number of days in a course, print all invalid emails, and print all students within a specific degree program.  

APPLICATION VERSION: Application Version 1.0, 06/27/2021

IDE: Xcode
Programming Language: C++
